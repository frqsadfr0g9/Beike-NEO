## 介绍 
### 实现的功能

**一、教务系统部分**  
1.课表：查看你的课表，支持正常学期和小学期。三击空白格子可以添加自定义课程。  
2.教务账户：登录本研一体的地方。受学校系统限制，登录过一段时间会被自动注销掉。   
3.考试：查看考试科目，时间，考场，距现在时间等信息。  
4.成绩：查询成绩，可快速计算加权。

**二、生活服务部分**  
1.无课教室：查看无课教室，数据来自学校官方，和逸夫楼一楼大屏完全一致。（注：请勿在连接vpn时查看，否则会加载不出来）  
2.网络服务：登录校园网账号，查看剩余流量和管理登录设备。（注：请在连接校园网时使用，流量和校外网络无法登陆。若想在连接流量或校外网络时使用，请使用WebVPN自行转换https://zifuwu.ustb.edu.cn ）  
3.流量查询：查看校园网流量使用与账单数据。  
4.电费查询：输入电表号即可查询电费，若每天都查询的话可以根据你的数据绘图。（注：电表号贴在宿舍电表箱上，为8位数字）  
5.WebVPN：在输入框内键入你想访问的校内网站，按下回车后会通过图书馆的WebVPN创建可在校外访问的URL，复制到浏览器即可一键回校。（注：当然也可以把WebVPN转为校内链接，不过我相信没有人会这么做）

**三、"其他"页面**  
1.配色方案：可切换亮色模式/暗色模式/跟随系统  
2.调色板：第1个颜色为跟随安卓系统的自动莫奈取色，其余颜色针对不支持莫奈的设备设计，比如初次打开app后默认强调色是蓝紫色的系统。国产安卓！说的就是你！（注：第二个颜色是北科蓝）  
3.课程提醒：在距离上课前25分钟发送通知。  
4.假期模式：一键删除缓存课表并禁用小组件，在来年开学手动获取课表时将被自动关闭。

**四、附加功能**  
1.桌面小组件：本功能支持安卓，ios与macos，小组件在有课时显示本节课的名称，时间和地点以及授课老师，在无课时显示“今日课毕”，在周末时显示周末祝福，在开启假期模式后显示假期祝福。

## 使用方法

1. 请[**前往此页面**](https://github.com/frqsadfr0g9/Beike-NEO/releases)下载适合你的操作系统的程序文件。
2. 安卓版请下载apk并且无视风险强行安装。
3. Windows版请下载zip并且解压到任意一个你认为合适的文件夹内，双击beike_neo.exe运行。
4. Linux版使用AppImage即可。
5. iOS版需要自签，巨魔或越狱才能安装，由于财力有限故不会上架app store，一年六百块钱耶，我怎么会掏得起！
6. macOS版右键打开dmg后将应用拖放至Applications文件夹内即可在程序坞见到它。若弹窗提示“无法打开Beike NEO，因为无法确认开发者身份“，那么请移步至设置➡️隐私与安全性，划到下面，点击允许任何来源的应用程序，如果有确认是否运行Beike NEO的地方，请点击仍要运行。
